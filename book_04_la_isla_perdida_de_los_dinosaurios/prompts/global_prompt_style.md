# Prompt global de estilo

Ilustración infantil animada premium para cuento ilustrado y libro de colorear, niños exploradores y dinosaurios bebés con poderes mágicos, aventura de isla perdida, mapas, montañas, ríos, lagos, trampas antiguas, guardianes de piedra y musgo, rescate familiar, tono emocionante y seguro para niños de 5 a 10 años, formas redondeadas, expresiones grandes, composición clara, no terrorífico.

## Versión color

Usar verdes vivos para la isla, azules brillantes para agua y energía, dorados para puertas familiares, luces verdes mágicas para trampas antiguas y azul espacial para el rescate final.

## Versión coloreable

Dibujo lineal limpio en blanco y negro, contornos gruesos, espacios amplios para colorear, trampas caricaturescas, guardianes simpáticos aunque imponentes, sin sombras complejas.

## Evitar

- violencia fuerte
- sangre
- heridas gráficas
- miedo excesivo
- oso terrorífico
- guardianes oscuros
- trampas realistas peligrosas

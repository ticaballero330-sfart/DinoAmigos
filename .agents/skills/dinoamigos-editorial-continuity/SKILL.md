# DinoAmigos Editorial Continuity

Use this skill for narrative and editorial work in the DinoAmigos project.

## Required reading

Read only what is needed for the task, but start with:

- `AGENTS.md`
- `README.md`
- `bible/00_saga_overview.md`
- `bible/07_editorial_rules.md`
- `bible/02_characters_master.md`
- `bible/continuity_matrix.md`
- `bible/editorial_checklist.md`

For a specific book, also read that book's:

- `README.md`
- `story/page_by_page.md`
- `story/continuity_notes.md`
- `story/synopsis.md` when the global arc matters

## Workflow

1. Identify the book, page, scene, or story layer being edited.
2. Confirm timeline state: characters present, powers available, objects active, Rexxie's mark status.
3. Preserve the target audience: ages 5 to 10, short sentences, clear emotion, safe magical adventure.
4. Keep each page visually illustratable with one clear action.
5. Validate against `bible/editorial_checklist.md` before finishing.

## Editorial rules

- Keep 32 pages per book unless the user explicitly asks otherwise.
- Preserve normalized names and relationships.
- Keep danger soft, playful, magical, or comic.
- End each book emotionally and keep the next-book hook gentle.
- Do not turn the saga into hard science fiction, horror, dark drama, or realistic combat.
- Do not redesign characters; flag visual contradictions only when they affect story continuity or prompts.

## Output standard

When reviewing, lead with concrete issues and file/page references when available. When creating, match the existing book structure and vocabulary.

# Prompt global de estilo

Ilustración infantil animada premium para cuento ilustrado y libro de colorear, niños exploradores y dinosaurios bebés con poderes mágicos, búsqueda de una planta secreta, marca alienígena roja suave, mapas verdes y dorados, guardianes dinosaurio, retos mágicos, T-Rex robots cómicos, jardín vivo de plantas luminosas, expresiones grandes, formas redondeadas, composición clara, aventura de sanación para niños de 5 a 10 años, no terrorífico.

## Versión color

Usar verdes y dorados para la Planta Secreta, rojo suave para la marca de Rexxie, azules para energía alienígena y agua, violetas suaves para Sombrín y colores alegres para guardianes y robots.

## Versión coloreable

Dibujo lineal limpio en blanco y negro, contornos gruesos, espacios amplios para colorear, personajes grandes, tecnología y plantas con formas simples, sin sombras complejas.

## Evitar

- sangre
- heridas gráficas
- robots violentos
- miedo intenso
- villanos oscuros
- plantas terroríficas
- lenguaje visual adulto
